# Doctrine Category Products

A simple PHP project using **Doctrine ORM** to manage **categories** and **products**, demonstrating:

- Fetching categories with products
- Filtering products by price using **Criteria**
- Using **ArrayCollection** methods
- Displaying results in HTML

## Features

- Filter products above a given price
- Order products by price descending
- Limit number of products displayed per category
- Minimal and clear code for learning Doctrine ORM

## Requirements

- PHP >= 8.0
- Composer
- Doctrine ORM
- MySQL or MariaDB
